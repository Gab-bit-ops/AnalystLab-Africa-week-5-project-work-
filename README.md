# 🏦 Bank Marketing Campaign Analysis

A end-to-end data analysis project investigating customer subscription behaviour for a Portuguese bank's term deposit marketing campaign.

## 📌 Project Overview
This project analyses a bank marketing dataset of **11,162 customers** to identify the key factors that influence whether a customer subscribes to a term deposit. The analysis follows a complete data pipeline — from raw data ingestion to interactive dashboard and business reporting.

## 🛠️ Tools Used
- **MySQL Workbench** — Data import, cleaning, and exploratory analysis
- **Power BI Desktop** — Interactive dashboard and KPI visualizations

## 📊 Key Findings
- Overall subscription rate of **47.38%**
- Customers aged **60 and above** had the highest subscription rate at 77%
- **Students and retired** customers were the most likely to subscribe
- **Cellular contact** was the most effective communication channel
- **74.6%** of customers had never been contacted in a previous campaign


## 🔍 Dataset
- **Source:** Portuguese Banking Institution — Direct Marketing Campaign
- **Records:** 11,162 rows | 17 attributes
- **Target Variable:** `deposit` (yes/no)
